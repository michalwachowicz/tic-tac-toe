# Tic Tac Toe

> An unbeatable tic tac toe game with the minimax algorithm

## Built With

- HTML
- SASS
- JavaScript

## The Minimax Algorithm

The game uses minimax algorithm to prevent you from winning the game 😈

The algorithm evaluates the possible moves based on the player's turn, then it chooses the move with maximum score for the AI and minimum score for the human player. Minimax will never let you win, so the game will always end up as a draw or a loss.
